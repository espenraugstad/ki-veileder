# KI - veileder

En KI-veileder bygd med følgende teknologier:

* React
* Typescript
* Vite
* Tailwind
* npm

## Installasjon

Installeres ved å klone repoet og bruke standard npm kommandoer:
```
npm install
```

Dev server kan kjøres med kommandoen:

```
npm run dev
```

## Merknad

Slik som spørsmålene opprinnelig var poengsatt så er det ikke mulig å oppnå resultatet "USIKKER". For å teste at det faktisk er mulig å vise dette resultatet har jeg lagt til et ekstra svaralternativ på spørsmål 2 som gir 0 poeng og kun er ment for testing. Det skal derfor være mulig å få dette resultatet ved å velge svaralternativene som inneholder "usikker" på alle spørsmålene.
